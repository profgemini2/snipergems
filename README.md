# snipergems

find gems before they trend.

snipergems is a lightweight on-chain signal tracker designed to surface early-stage opportunities before they gain mainstream attention.

## features

- early signal detection (volume, wallet activity)
- simple scoring system for potential gems
- noise filtering for low-quality tokens
- fast and minimal setup

## usage

npm install  
node src/index.js
